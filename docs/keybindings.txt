This is the set of default hotkeys for GIMP. Note: Gimp allows 
menu keybindings to be dynamically re-assigned. So if you dont
like these, they are easy enough to change. GIMP also comes with
a menurc that will implement PS style keybindings, if you are used
to PS, this may prove helpful.

version 1.0 Nov 7 Adrian Likins <adrian@gimp.org>
