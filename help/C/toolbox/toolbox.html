The ToolBox

Probably the window that is most central to the use of The GIMP is the ToolBox.

Figure 3-1. The ToolBox

The ToolBox consists of the menu bar with the entries File, Xtns and Help, the tools buttons which are the set of square buttons in Figure 3-1, the color selector which is at the bottom left, and the tools status indicators which are at the bottom right.

Almost all the tools in the toolbox have options which you can configure to define how the tool operates. You can access these Tool Options either by double clicking on the tool's icon.
