Stroke Path

Stroke (draw a line) along the currently active path with the current painting tool. The stroke line is drawn with the center of the paint tool on the center of the path. This function is the same as clicking the icon.

For example if the paint tool selected is the the normal "Paintbrush" then this will be used to create the line. If this tool has any options set, e.g use a gradient or fade out, then this will be honoured when the path is stroked. This applies to any of the modes that this painting tool accepts.

The following paint tools work when stroking a path. They need to be the currently active tool to work, if no paint tool is selected then the "Paintbrush" tool is used.

  • Pencil

  • Paintbrush

  • Eraser

  • Airbrush

  • Clone

  • Convolver

  • Dodge or Burn

  • Smudge
