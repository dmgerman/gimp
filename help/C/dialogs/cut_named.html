Cut Named

Lets you cut the current selection to a named buffer. You can cut several parts to different buffers by giving them different names. Later on you are able to paste a selected buffer by invoking Image Menu > Edit > Buffer > Paste Named.... If you don't specify a selection, the whole active layer is cut.

Additional Information

Default Keyboard Shortcut: Shift-Ctrl-X
