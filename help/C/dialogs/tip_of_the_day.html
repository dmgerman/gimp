Tip of the Day

Will display a useful tip each time you start Gimp. You can navigate the tip dialog by clicking on Previous and Next Tip. A good tip can be to disable it since you may find it annoying. Just uncheck Show tip next time and it will be vanished from Gimp's startup ;-).

If you are a good Gimp user then it can be a good tip to submit useful tip(s) to the Gimp developers.
