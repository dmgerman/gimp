Info Window

The General Tab folder

Dimension (w x h):
Show the image height and width both in pixels and in the current unit
Resolution:
Shows the current resolution
Scale Ratio:
Shows the current zoom factor
Display Type:
Shows the current image mode
Visual Class:
Shows the visual class of your display
Visual Depth:
Shows the visual depth of your display

The  Extended Tab folder

Shows you the current pixel value while you moving the pointer over the image without pressing any buttons. I.e it's a color picker that not affect the current color but instead gives you the pixel value regardless of the current tool. For information how to interpret the pixel value see the Color Picker tool.
