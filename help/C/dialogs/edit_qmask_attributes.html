The Edit Qmask Attributes Dialog

Allows you to alter the name, color and fill opacity of the quick mask channel.

The fill opacity is mainly useful if you want to set how much of the underlaying image you want to see, when you work with quick mask that you later on will turn into selections. If you set it to high you will not be able to see the image that you are working with. Changing the color is most of the time only useful if the image you are working with is mostly red.
