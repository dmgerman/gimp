Scale Layer

Will scale the layer content,  and also make the layer boundary either size smaller or bigger. The difference between Scale Layer and Set Layer Boundary Size is that Set Layer Boundary Size will enable you to have a smaller or bigger layer without scaling the layer content (i.e it will only add some space to on around the layer or clip the layer). NOTE: Scale layer will only scale the currently active layer, scale image will scale all layers with in the image.

You will set the new layer size either by altering the size or ratio. You have have exactly the same options as in create new layer.
